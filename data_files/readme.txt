How to build tables and replicate the data of our F1 Database:

Tables Creation:
1.Open pgadmin create a new database by clicking on the postgre server.
2.Once the database is created, right click on the database and click on restore.
3.Browse and select create.sql file and click restore.
4.This will create the schema and the tables inside it.

Loading data into tables:
1.On the same db right click and select restore.
2.Browse and select load.sql file and click restore.
3.This will populate the data into the tables.
